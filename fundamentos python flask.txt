1. Explicación del Código (Línea por Línea)
from flask import Flask: Aquí estás importando una clase llamada Flask. En Python, una clase es una plantilla para crear objetos. Estás trayendo las herramientas necesarias para que tu script sepa cómo ser un servidor web.

app = Flask(__name__): Creas una instancia (un objeto individual) de esa clase. __name__ es una variable especial de Python que le dice a Flask dónde buscar archivos (como tus carpetas de imágenes).

@app.route('/'): Esto se llama un decorador. Su función es conectar una dirección de internet (URL) con una función de Python. El '/' significa la página principal (la raíz).

def inicio():: Es una función estándar de Python. Flask la ejecutará automáticamente cuando alguien visite la raíz de tu sitio.

return "<h1>...</h1>": Es lo que la función le devuelve al navegador. El navegador recibe este texto en formato HTML y lo dibuja en la pantalla.

if __name__ == '__main__':: Esta línea verifica si estás ejecutando el archivo directamente. Si es así, activa el servidor con app.run(debug=True). El debug=True es vital: hace que el servidor se reinicie solo cada vez que guardas un cambio.

¿Por qué hicimos esto? (Fundamentos)
Cascading Style Sheets (CSS): Se llama "Cascada" porque los estilos se aplican de arriba hacia abajo. Lo que pongas al final puede sobrescribir lo de arriba.

Variables (:root): Esto es de arquitecto profesional. Si mañana decides que tu verde ya no es bosque sino esmeralda, solo cambias una línea de código arriba y toda la página se actualiza sola.

Pseudoclases (:hover): Es lo que le da "vida" a la web. Google Ads valora mucho la Experiencia de Usuario (UX). Si la página reacciona cuando el usuario interactúa, se quedan más tiempo, y Google te posiciona mejor.

Paso 3: Conexión con tu HTML
Para que esto funcione, asegúrate de que en tu index.html (el que hicimos en el paso anterior) esté esta línea dentro del <head>:

<link rel="stylesheet" href="{{ url_for('static', filename='css/estilo.css') }}">

Ese código entre llaves {{ ... }} es Jinja2 (el lenguaje de plantillas de Python). Le dice a Flask: "Oye, busca en la carpeta 'static' el archivo que está en 'css/estilo.css' y conéctalo".
